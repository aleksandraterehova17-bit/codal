# codal
Expense Tracker — Трекер расходов

Автор: Терехова Александра Антоновна  

Описание программы

Expense Tracker — это приложение с графическим интерфейсом (GUI) на Python (библиотека Tkinter), которое помогает пользователям:

- Добавлять и записывать ежедневные расходы  
- Фильтровать траты по категориям (Еда, Транспорт, Развлечения и др.)  
- Фильтровать расходы по периоду (дата от / дата до)  
- Автоматически подсчитывать общую сумму расходов за выбранный период  
- Сохранять все данные в файл `expenses.json` (при повторном запуске загружать их)  
- Удалять ненужные записи  

Программа проверяет корректность ввода: сумма должна быть положительным числом, дата — в формате `ГГГГ-ММ-ДД`.
